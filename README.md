# Evolution Swing: 進化するブランコ棒人間

## 🎯 このプロジェクトについて

この教材は、「進化計算（遺伝的アルゴリズム）」を使って、2Dの棒人間がブランコを**自力でこげるようになるまで**を学ぶ演習用プロジェクトです。

最初はまったく動けなかった棒人間が、世代を重ねることでだんだん「うまく揺れる」ようになります。

- 使用言語：Python（初級〜中級）
- 使用ライブラリ：`pybox2d`, `pygame`
